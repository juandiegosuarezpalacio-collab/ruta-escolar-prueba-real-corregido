PRUEBA REAL - RUTA ESCOLAR

1. Abre ESTA carpeta completa en VS Code.
2. Usa Live Server sobre index.html.
3. No abras el archivo con doble clic.
4. Verifica estas rutas en el navegador:
   /data/estudiantes.json
   /data/barrios.json

Si ves JSON, la carga esta bien.
Si ves HTML o la pagina principal, el servidor esta mal apuntado.

Tambien puedes usar Python:
python -m http.server 5500
Luego abre:
http://localhost:5500/
